# Django Practice: REST API & PostgreSQL Integration

A foundational Django project focused on learning backend architecture, relational database management, and RESTful service implementation.

## 🚀 Project Overview
This repository serves as a practice for implementing:
- **Django Core:** Apps, Models, and MVT architecture.
- **REST Framework:** Building API endpoints using Django REST Framework (DRF).
- **PostgreSQL:** Transitioning from SQLite to a production-grade database.

## 🛠 Tech Stack
- **Language:** Python 3.x
- **Framework:** Django 5.x
- **API:** Django REST Framework (DRF)
- **Database:** PostgreSQL
- **Tools:** venv, Git, psycopg2
